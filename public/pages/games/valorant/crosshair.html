Valorant Crosshair Codes

Danh sách các crosshair code được sử dụng bởi các game thủ chuyên nghiệp và streamer. Copy code và paste vào game để sử dụng.

Cách sử dụng Crosshair Code

  1. Copy code crosshair bạn muốn sử dụng
  2. Mở Valorant và vào Settings
  3. Chọn tab Crosshair
  4. Click vào nút Import Profile
  5. Paste code vào và nhấn Import
